## 第一题（AC）

题目描述：

    参考LeetCode-453最小移动次数使数组相等

输入输出格式：

    输入第一行n表示数组长度
    第二行n个数字表示数组元素大小，以空格间隔

    输出一个数字表示结果

测试用例：

    输入：
    3
    1 2 3

    输出：
    3

## 第二题（AC）

题目描述：

    给定一个字符串S和n个数字变换（a,b）每个变换是将字符串里的数字a变为b，
    如果字符串里不存在a，则不变
    输出全部操作完后的字符串


输入输出格式：

    输入第一行一串0-9组成的字符串s
    第二行一个整数n
    第三行开始的n行每行是两个数a,b代表数字a变成b
    1 <= |S| <= 1000000
    1 <= n <= 100000

    输出变换后的字符串

测试用例：

    输入：
    00011222233344
    2
    0 2
    2 3

    输出：
    33311333333344


难度：★☆☆☆☆
